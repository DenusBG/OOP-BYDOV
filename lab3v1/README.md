# Лабораторна робота №3: Спадкування

**Виконав:** Будов Денис

## Тема та мета роботи

**Тема:** Спадкування: основи.

**Мета:** закріпити теоретичні знання та отримати практичні навички використання механізму спадкування в C#. Створити ієрархію класів, використати абстрактні класи, перевизначення методів та продемонструвати поліморфізм.

---
## Опис виконання завдання

У ході лабораторної роботи було реалізовано ієрархію класів для представлення геометричних фігур.

1.  **Створено абстрактний базовий клас `Shape`**, який визначає загальну поведінку для всіх фігур. Він містить властивість `Name` та абстрактні методи `Area()` та `Perimeter()`, що слугують "контрактом" для класів-нащадків.

2.  **Створено два похідні класи, `Circle` та `Rectangle`**, що успадковують `Shape`. Кожен з них реалізує (`override`) методи `Area()` та `Perimeter()` відповідно до власних геометричних формул. Конструктори нащадків викликають конструктор базового класу за допомогою ключового слова `base` для ініціалізації імені фігури.

3.  У головному файлі `Program.cs` продемонстровано принцип **поліморфізму**: створено колекцію `List<Shape>`, яка містить об'єкти обох дочірніх типів. Програма в циклі обробляє всі фігури через єдиний інтерфейс `Shape` та знаходить фігуру з найбільшою площею.
